Simple Restaurant PWA apps with dummy data

![landing-page](readme.png)

## Description
This website built with:
* HTML
* CSS
* Webpack
* Workbox


## Run Local
* Clone this repo:

`$ git clone https://github.com/Fachrulmustofa20/RestaurantApp-PWA.git`

* Install dependencies

`$ npm install`

#### Development
`$ npm run start-dev`

#### Production
`$ npm run build`
